sujet mail NF17 Groupe 4
écrire les contraintes, les clef candidates, ex un restaurant a une seule carte a la fois et les deux dates ne se chevauchent pas

Tous les attributs où ce n’est pas spécifié NULLABLE sont NOT NULL.

Restaurant (#idRestaurant: int, nomRestaurant: string, adresse:string, ville:string, pays:string,)
	Contraintes: (nomRestaurant, adresse, ville, pays) clé candidate

Cuisinier (#idEmploye: int, spécialité: string, nom: string, prénom: string, dateNaissance: date, dateAnciennete: date, idRestaurant=>Restaurant: int)
	Contraintes: (nom, prénom, dateNaissance) clé candidate

Manager(#idEmploye: int, nom: string, prénom: string, dateNaissance: date, dateAnciennete: date, idRestaurant=>Restaurant: int) 
	Contraintes: (nom, prénom, dateNaissance) clé candidate

Serveur(#idEmploye: int, authorisationAccueil: boolean, nom: string, prénom: string, dateNaissance: date, dateAnciennete: date, idRestaurant=>Restaurant: int) 
	Contraintes: (nom, prénom, dateNaissance) clé candidate

Carte (#idCarte: int, nomCarte: string)

Commande(#idCommande: int, #idRestaurant=>Restaurant: int, dateCommande: date)

Menu(#idElement=>Element :int, nomMenu: string)

Element(#idElement: int)

Plat(#idElement=>Element: int, nomPlat: string, catégorie: string, entree: boolean, plat: boolean, dessert: boolean)

BoissonOfferte(#idElement=>Element: int, nomBoisson=>Boisson: string, volumeDecilitres: decimal, anneeProduction: int)
	Contraintes :
	- anneeProduction NULLABLE
	- volumeDecilitres > 0

Boisson(#nomBoisson: string, type: string)

Ingredient(#nomIngredient: string, liquide: boolean, solide: boolean)

PeriodeCarte(#idCarte=>Carte: int, #idRestaurant=>Restaurant: int, #dateDebut: date, dateFin: date)
	Contraintes:
	- dateDebut < dateFin
	- On ne peut pas modéliser la contrainte suivante: Un restaurant a une seule carte a la fois par période. C’est à dire que deux périodes de deux cartes différentes ne peuvent pas se chevaucher pour un même restaurant.

QuantiteElement(#idElement=>Element: int, #idCarte=>Carte: int, quantiteElement :int)
	Contraintes: quantiteElement > 0

PrixElement(#idElement=>Element int, #idCarte=>Carte: int, prixElement: decimal)
	Contraintes: prixElement > 0

AssocMenuPlat(#idMenu=>Menu: int,#idPlat=>Plat: int)

QuantiteIngredient(#nomIngredient=>Ingredient: string, #idElement=>Plat :int, quantiteIngredient: int)
	Contraintes: quantiteIngredient > 0
